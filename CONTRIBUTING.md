# Contributing to IdleClaw

Thanks for your interest in contributing to IdleClaw.

## Ways to contribute

- **Share your GPU** — Run the node agent to contribute inference capacity to the network
- **Report bugs** — Open an issue with steps to reproduce
- **Fix bugs** — Pick up an open issue and submit a PR
- **Improve docs** — README, SKILL.md, code comments

## Development setup

See the [README](README.md) for local development instructions. You'll need Node.js 18+, Python 3.11+, and Ollama.

## Pull requests

1. Fork the repo and create a branch from `main`
2. Make your changes and test locally
3. Keep PRs focused — one feature or fix per PR
4. Write clear commit messages describing _why_, not just _what_

## Code style

- **Frontend:** TypeScript, follows existing patterns. Run `npm run lint` before submitting
- **Python:** Standard formatting. Keep imports sorted
